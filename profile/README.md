## About

organization is home to the tooling, libraries, and utilities we rely on to ship

## What we build

| Area                    | Focus                                                                    |
| :---------------------- | :----------------------------------------------------------------------- |
| **Developer tooling**   | Focused CLIs and libraries that solve one problem well.                  |
| **AI & LLM**            | Agents, retrieval, and integration tooling for production AI.            |
| **Web3**                | dApp building blocks, SDKs, and on-chain utilities, primarily on Solana. |
| **UI & design systems** | Accessible, composable components built on shadcn/ui.                    |
| **Security**            | Audits, vulnerability research, and defensive tooling.                   |

## How we work

> **Ship fast, iterate faster** — small, focused releases over big-bang launches.
> **Production-grade from day one** — typed, tested, and documented.
> **Clear communication, no surprises** — readable code and honest docs.
> **Long-term partnership mindset** — we maintain what we ship.
